          КОМАНДНЫЙ ПРАКТИЧЕСКИЙ ТУР — ПРОГРАММНЫЙ МОДУЛЬ
          ДЕНЬ 3. Сопровождение цели с обходом препятствий

НАЗВАНИЕ КОМАНДЫ: Чижики
ФАЙЛ РЕШЕНИЯ:     Test7.py
Дата подготовки:  25 марта 2026

1. КРАТКОЕ ОПИСАНИЕ РЕШЕНИЯ
────────────────────────────────────────
• Автономный взлёт на высоту 1.8 м и начальный манёвр вправо (от сетки).
• Поиск и захват выбранной целевой фигуры по классу target_person и ID.
• Режим сопровождения (TRACK) с удержанием дистанции 1.5 м по /front_rangefinder/range.
• Реактивный обход препятствий класса obstacle (режим AVOID) при сближении ближе 0.8 м.
• Автоматическое восстановление режима TRACK после обхода.
• Таймаут миссии 180 секунд → автоматическая посадка.
• Полностью соответствует требованиям Дня 3 (разделы 4–5 PDF).

2. АРХИТЕКТУРА АВТОМАТА СОСТОЯНИЙ
────────────────────────────────────────
• WAIT_START   → ожидание старта
• SEARCH       → поиск/вращение при потере цели
• TRACK        → сопровождение целевого человека
• AVOID        → реактивный обход препятствия (остановка + предупреждение)

Переключение состояний происходит в detections_callback на основе:
- /vision/detections (target_person + obstacle)
- /front_rangefinder/range

3. ПОШАГОВЫЙ ПОРЯДОК ЗАПУСКА
────────────────────────────────────────
1. Запустить детектор TFLite:
   roslaunch clover detector_profile.launch \
     enabled:=true \
     node_name:=detector_tflite \
     config_path:=/home/pi/catkin_ws/src/clover_yolo/config/profiles/ВАШ_ПРОФИЛЬ.yaml

2. Запустить миссию:
   rosrun [ИМЯ_ВАШЕГО_ПАКЕТА] Test7.py

   (или через launch-файл — см. ниже)

3. Дрон выполнит всю миссию автоматически и совершит посадку.

4. ИСПОЛЬЗУЕМЫЕ ROS-ИНТЕРФЕЙСЫ 
────────────────────────────────────────
Подписки:
• /vision/detections          (clover_yolo/DetectionArray)
• /front_rangefinder/range    (sensor_msgs/Range)

Сервисы:
• get_telemetry
• navigate
• land

Параметры (в коде, можно вынести в launch):
• target_dist = 1.5 м
• target_altitude = 1.8 м
• safe_dist = 0.8 м (безопасная дистанция до obstacle)

6. СООТВЕТСТВИЕ ТРЕБОВАНИЯМ РЕГЛАМЕНТА ДНЯ 3
────────────────────────────────────────
• Используется target_person + obstacle 
• Удержание дистанции по дальномеру 
• Обход препятствий (AVOID) 
• Восстановление сопровождения после потери 
• Автономный взлёт, возврат и посадка 
• Нет ручного вмешательства после старта 
• Предсказуемая логика автомата состояний 

7. ДОПОЛНИТЕЛЬНАЯ ИНФОРМАЦИЯ
────────────────────────────────────────
• Высота полёта:               1.5 м
• Заданная дистанция до цели:  1.5 м
• Минимальная безопасная дистанция: 0.8 м (можно уменьшить до 0.5 м)
• Таймаут миссии:              180 с
• Логирование:                 target_id, состояние, дистанция (с throttle)

Решение готово к запуску в целевой среде без редактирования кода.
Все изменения параметров — только через launch-файл.
